# 🏟️ Polymarket Arena

## Real-Time AI Trading Competition Dashboard

5 AI trading bots compete in a 48-hour battle for supremacy on prediction markets.

### 🚀 Live Demos

- **Desktop Version**: Full-featured dashboard with detailed analytics
- **Mobile Version**: Touch-optimized interface for phones and tablets

### 🤖 Competing Bots

1. **Cross-Market Divergence** - Detects price differences across prediction markets
2. **Sharp Wallet Copy** - Mirrors trades from high-performing wallets  
3. **Wikipedia Velocity** - Detects breaking news via Wikipedia edit spikes
4. **LP Withdrawal Detection** - Monitors smart money liquidity exits
5. **Economic Data Positioning** - Positions before scheduled data releases

### 📊 Features

- **Real-time leaderboard** with live ROI tracking
- **Performance analytics** with risk vs return analysis
- **Live trade feed** showing bot decisions in real-time
- **Market opportunities** cross-bot intelligence sharing
- **Mobile-optimized** interface with touch-friendly controls

### 🏆 Competition Format

- **Duration**: 48 hours
- **Starting Balance**: $10,000 per bot
- **Mode**: Paper trading (no real money)
- **Winner**: Highest ROI after 48 hours
- **Updates**: Every 8-10 seconds

### 🛠️ Built With

- **Streamlit** - Web framework
- **Plotly** - Interactive charts
- **Pandas** - Data manipulation
- **Real-time updates** - Auto-refreshing dashboard

### 🎮 How to Run Locally

```bash
pip install -r requirements.txt

# Desktop version
streamlit run app.py

# Mobile version  
streamlit run mobile.py
```
